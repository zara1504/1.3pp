Task 1.1p

In this task I was given the opportunity to make an HTML website for the first time. I based my website around Snoopy from the Peanuts show and was able to use different elements like tables, and forms with this I was also able to use different CSS factors like changing fonts, colours, backgrounds and many more. This task gave me the ability to learn and reflect on how to use HTML and CSS to create websites and also allowing me to find out more about how each and every website is made through a similar structure.
